serialization.settings_tpl module¶

mp_to_word

Настройка для генерации результирующего файла. Все блоки(разделы) печатного варианта мп, должны следовать в определенном порядке Для кадого раздела указываться

tpl: шаблон для рендеринга

pos_doc: позиция блока в результирующе печатном варианте

class: класс для получения источника данных

clear: удалять узлы, после прохождения по xml, данный атрибут на дынный момент важен, пока не придумано обхода,

к примеру элемент ProvidingPassCadastralNumbers - скажем таки размазан по всему файлу xml
